# Getting Started

## Welcome to First Movers!

Below you will find a brief guide to help you get started and start winning on TikTok Shop. First Movers is built on one idea: whoever posts a product first wins the views, the sales, and the ad spend. This guide gets you set up to be that person.

## The GMV Max Era

TikTok Shop moved to a GMV Max model, an automated ad system that pushes budget behind products that are converting. Brands set their targets, TikTok finds the content driving sales, and it pours ad spend into whatever is working.

For affiliates, this means the game is speed. Every time a top brand drops a new product, variant, flavor, or bundle, there is a short window where GMV Max budgets are hunting for content. Catch that window and your commissions stack. Miss it and you are posting into a saturated feed where everyone is already selling the same thing.

While everyone else is manually checking Kalodata and Fastmoss for new winners that take days to update, First Movers already sent the alert.
