# Navigating the Server

The server is organized so you can find what you need fast. Here's the layout.

<figure><img src="../.gitbook/assets/73056-sidebar.png" alt="First Movers server sidebar"><figcaption>The full server sidebar</figcaption></figure>

Everything is grouped into categories on the left. The three that matter most when you're getting started:

## WELCOME

Where you get set up and oriented. Start here on day one.

<figure><img src="../.gitbook/assets/73126-welcome.png" alt="WELCOME category"><figcaption></figcaption></figure>

* **start-here** — your first stop, the quick orientation
* **how-to-use** — how to read alerts and use the retail buttons
* **get-access** — connect your account and unlock the server
* **reviews** — wins from members

## PRODUCT ALERTS

Where the actual alerts land. This is the core of the service.

<figure><img src="../.gitbook/assets/73157-product-alerts.png" alt="PRODUCT ALERTS category"><figcaption></figcaption></figure>

* **trending-products** — products already racking up sales, posted early
* **Category channels** — health-and-wellness, beauty-and-personal-care, food-and-beverages, womens-fashion, mens-clothing, lifestyle-gear-and-home, sports-and-outdoors, maternity-and-kids, pets, footwear, others

Each category channel only fires alerts for products in that lane. You pick which ones ping you in **select-roles** — covered next.

## Other Categories

* **CURRENT SHOPS** — the list of retailers we match products against
* **SUPPORT** — open a ticket if you need help
